2019 8-6A Basketball Projections

The basketball forecast uses win loss record, and strength of schedule to create a beginning ELO. ELO is a type of rolling ranking that changes after every game but never the total ELO points never change. After each game the winner takes ELO points from the losing team, and the bigger the upset, the bigger the ELO tradeoff. The simulation runs through every game of the season, and determines the winner based on ELO match up, and random generated numbers. The simulation is then ran 10,000 times to get the forecasted results, which change after every game.
